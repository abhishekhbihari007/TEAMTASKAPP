# Team Task Manager

A full-stack project orchestration platform designed for team collaboration.

## Features
- **Project Management**: Create projects and manage team members.
- **Task Tracking**: Create, assign, and track task progress in real-time.
- **Role-Based Access**: Specialized admin and member roles.
- **Dashboard**: Overview of tasks, status, and overdue items.

- **Frontend**: React, Tailwind CSS
- **Backend**: Node.js, Express, MongoDB
- **Authentication**: JWT, Bcrypt Password Hashing
